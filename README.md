# A Mazer Pather

The zillionth maze pather made with the A* algorithm, C, and Raylib
<br/><br/><br/>

## Getting Started

### Dependencies

* Raylib
* Cmake (or preferred compiler)

### Installing

* Make a build/ and run ```Cmake .```inside build/
* Run ```make .``` inside build/

### Executing program

* Default is ```./maze_pather``` to execute program.
